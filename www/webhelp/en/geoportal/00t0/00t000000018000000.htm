Custom Metadata Profiles Overview

The Geoportal Server provides publishers with the ability to create metadata that conforms to any standard using an online Create Metadata form. To access the form, a user with publisher permissions logs in, selects the Administration tab, selects the Add link at the top of the page, and then selects Use dedicated editor to create metadata manually from the choices on the Add page. The resulting form supports several popular metadata standards out-of-the-box. When these standards do not suit your organization's needs, the Geoportal Server can be customized to support other metadata standards or create your own for use within your organization.
