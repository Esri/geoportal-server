What's New

What's New with the Geoportal Server

The Geoportal Server provides a number of enhancements and new functionality. For specific information on issues resolved with the Geoportal Server or Geoportal extension 10, see the Release Notes. This section describes what's new for the Geoportal Server and Geoportal extension 10:

  • Expanded Platform Support
  • Discovery Enhancements
  • Publishing Enhancements
  • Interoperability Enhancements
  • Expanded Documentation

Expanded Platform Support

The Geoportal Server was tested in additional software environments. These are highlighted below. To see the complete list of supported system requirements, see Preinstallation Requirements:

  • Operating System: Windows 7
  • Database: PostgreSQL 8.4
  • Browsers: Chrome and Safari
  • ESRI Software: ArcGIS Server Enterprise Standard 10, ArcGIS Desktop 10, ArcGIS API for JavaScript version 2.0, and all Geoportal Clients for ArcGIS Desktop have been rewritten to support ArcGIS Desktop 10
  • Support of Javascript API version 2.0 in geoportal search and preview maps
  • Migration utility: to migrate Geoportal 9.3.x databases to the Geoportal Server database schema

Discovery Enhancements

Several enhancements have been implemented that enhance the user experience and increase the capabilities for discovering resources through the geoportal. The following features are new:

  • Federated Searching: The Geoportal Server allows users to search multiple registered catalogs at once, simultaneously.
  • Search ArcGIS Search Services: Users have the ability to search ArcGIS Server search services. A search service is new at ArcGIS Server version 10 and makes a searchable index of an organization's GIS content available for discovery. Visit the ArcGIS Server Search Services webhelp topic for more information on search services.
  • Provide feedback on registered content: Users now have the ability to leave comments about a resource. This feedback can improve the quality of the resources in the geoportal over time, and helps other users evaluate the resources of a catalog. For more information, see How to Leave a Resource Review.
  • Rate registered content: Along with providing feedback, users can rate a resource with either a thumbs up or thumbs down. For more information, see How to Leave a Resource Review.
  • Data download capability uses new ArcGIS server clip, zip and ship tool: ArcGIS server has an improved clip and ship tool at version 10, and the geoportal's data download customization takes advantage of this new tool. Visit the ArcGIS Server GP Service example: Clip and ship webhelp topic for more information about the clip and ship tool. For more information on how to configure the geoportal's data download customization, see DataDownload Tab.
  • Geoportal Search Widget for Silverlight: allows users to search geoportals from within a Silverlight-based map viewer. For more information, see Geoportal Search Widget for Silverlight.

Publishing Enhancements

The Geoportal Server has replaced its desktop Harvesting Tool and Harvesting Service with complete server-side synchronization capability handled by the geoportal web application alone. This simplifies registration of metadata documents synchronized from remote catalogs or service endpoints. For more information, see Register a resource on the network.

Interoperability Enhancements

The following interoperability enhancements have been made:

  • Support for OGC CS-W has been extended with the ISO application profile and INSPIRE discovery service.
  • Enhanced INSPIRE metadata support.
  • All Geoportal Desktop Clients are fully localizable.

Expanded Documentation

The format for the help documentation of Geoportal Server has been divided into two sections. The User Help section is for geoportal end users, and the Developer Help is for developers and system administrators. The Geoportal Server help documentation is available from two sources:

  • Context-sensitive help is included with the Geoportal Server installation and locally accessible
  • The latest content is always available online.
